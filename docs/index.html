Raven

Raven platform is a platform to provide a reference implementation and testing tools for the interoperability between case management system (CMS) and electronic death registration system (EDRS)

Github Demo Docs

Features

Ingest API

Import your MDI case data using the CSV format and Raven will map your data to FHIR

Dashboard

You can browse and view your cases via Raven's Case Summary Dashboard.

FHIR Service

Stores Case Data to FHIR Database

Export API

You have the ability to send cases to the respective EDRS Systems by using the Raven Export functionality.
